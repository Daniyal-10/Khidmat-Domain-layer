# 00-governance

## Purpose
Governance & Foundations. Contains project identity, constitution, and universal glossary.

## Scope
- **In Scope**: Project Overview, Constitution, Foundation, Vision, Philosophy, Principles, Glossary.
- **Out of Scope**: Implementation details, Methodology, Domain models.

## Canonical Documents
- `PROJECT_OVERVIEW.md` (v1.0, frozen) — sole conceptual authority.
- `PROJECT_OVERVIEW_BLUEPRINT.md` — the interview/authoring guide that produced it; historical, not itself authoritative content.
- `CONSTITUTION.md` (v1.0, frozen) — sole governance authority.
- `VISION.md`, `GLOSSARY.md` — synchronized against the Overview.
- `FOUNDATION.md` — not yet authored; scoped narrowly to a ratification record (see its status note) to avoid duplicating the Overview or Constitution.
- `PHILOSOPHY.md` — not yet authored; scoped as extended commentary distinct from the Overview and Constitution (see its status note).
- `PRINCIPLES.md` — not planned as a separate document; Constitution Article II already fulfills this role (see its status note).
