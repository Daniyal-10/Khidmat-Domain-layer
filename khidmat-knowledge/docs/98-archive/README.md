# 98-archive

## Purpose
Archive. Deprecated normative documents.

## Scope
- **In Scope**: Old versions of Normative docs.
- **Out of Scope**: Current working documents.

## Upstream Dependencies
- All layers

## Downstream Consumers
- None

## Categories
- `superseded-reviews/` — valid, completed reviews whose recommendations have since been fulfilled by later canonical work. Retained for institutional memory.

Every archived document carries its own in-file banner explaining its specific disposition. Historical content within archived files is preserved unmodified; only file location changes.

## Retired Categories

Two categories previously existed here and were deleted after review confirmed they contained no unique canonical information, per the Constitution's discipline that no methodology or architecture document may treat unratified prior work as authoritative input regardless of where it is stored:

- **`engineering-layer/`** (schema-first ontology-engineering contracts, migration plans) — fully superseded by the Overview's ontology-first mandate (Ch5.1) and Constitution Article XV; recoverable from git history at commit `d28d17e` and ancestors if ever needed for reference.
- **`invalid-lifecycles/`** (the Business Master Plan's review/resolution/certification trail for content that was never authored) — the four genuinely-grounded discovery topics they surfaced were merged into `docs/01-methodology/BUSINESS_MASTER_PLAN_BLUEPRINT.md` §7 before deletion; the remainder analyzed nonexistent content and is recoverable from git history if ever needed.
